立即在您的 IDE、CLI 或云端开始使用 Codex

ChatGPT Plus、Pro、Business、Edu 和 Enterprise 套餐均包含 Codex。将 Codex 与 ChatGPT 订阅配合使用，即可访问最新的 Codex 模型和功能。

## 设置

### IDE 扩展

#### Cursor

点击链接：

[cursor:extension/openai.chatgpt](cursor:extension/openai.chatgpt)

会自动打开 cursor 的扩展市场，点击安装。

#### VS Code

点击链接：

[vscode:extension/openai.chatgpt](vscode:extension/openai.chatgpt)

会自动打开 vscode 的扩展市场，点击安装。

### CLI 安装

安装 Codex CLI：

```bash
npm install -g @openai/codex
```

安装后检测：

```bash
$ codex --version
codex-cli 0.101.0
```
